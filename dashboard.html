Setup Guide

  1. Go to developers.facebook.com and create a Meta App
  2. Add the WhatsApp product to your app
  3. Go to WhatsApp → API Setup to find your Access Token and Phone Number ID
  4. Paste them into the form above and click Save
  5. Copy the Webhook URL shown above
  6. Go to WhatsApp → Configuration → Edit Webhook
  7. Paste your Webhook URL and Verify Token, then click Verify and Save
  8. Subscribe to the 'messages' webhook field
